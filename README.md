# interfacegasesideais
Interface gráfica desenvolvida com o intuito de auxiliar os alunos de ensino médio no estudo dos gases ideais.
A interface permite a realização de cálculos e possui os principais as principais leis e equações que descrevem os gases ideais, a saber:
-Equações de Estado escritas em diferentes formas;
-Lei Geral dos Gases;
-Lei de Boyle;
-Lei de Charles;
-Lei de Charles e Gay-Lussac;
-Cálculo da densidade de gases.
